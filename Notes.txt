Bash:
export JAVA_HOME="$(brew --prefix openjdk@17)/libexec/openjdk.jdk/Contents/Home"
export PATH="$JAVA_HOME/bin:$PATH"

brew install --cask react-native-debugger
open -a "React Native Debugger" --args --port 8081
Open your app in Expo Go.
Shake the device (or press Ctrl+D in Android emulator / Cmd+D in iOS simulator).
Choose Debug Remote JS.
Expo will open Chrome DevTools — close that.
In React Native Debugger, press:
Cmd+T → set port to 8081.
